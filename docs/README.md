# Demo media

Place the screen recording here as `demo.gif` (15–30 seconds).

Recommended capture:
1. Open two browser windows side-by-side, both at the live URL.
2. Add a card in window A → it pops into window B.
3. Drag a card from To Do → Done in window B → it animates in window A.
4. Close one window → presence count drops in the other.

Tools: ScreenToGif (Windows), Kap (macOS), `peek` (Linux). Aim for under 5 MB so GitHub renders it inline.
